# Shopify → Monday.com Order Sync Automation

This project automates the synchronization of Shopify orders into Monday.com using Make.com.

It eliminates manual data entry by automatically creating and updating items in Monday.com whenever a new order is placed in Shopify.

---

## 🚀 Overview

This automation captures order data from Shopify and processes it through multiple routes to:

- Create new items in Monday.com
- Update existing records
- Handle multiple order conditions
- Structure data using JSON parsing
- Ensure accurate and real-time synchronization

---

## ⚙️ How It Works

1. A webhook listens for new Shopify orders  
2. Data is routed based on conditions  
3. JSON module processes structured data  
4. Orders are checked in Monday.com  
5. Items are created or updated accordingly  
6. Iterators handle multiple line items  
7. Final data is synced and stored
